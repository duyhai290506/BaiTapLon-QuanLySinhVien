# Project1 bai_tap_lon-quan ly sinh vien
project môn nhập môn lập trình c/c++(quản lý sinh viên)
các chức năng chính+ xem tổng quát gpa,thêm,sửa xóa thông tin sinh viên,tìm kiếm và sắp xếp 1 danh sách. Chuẩn hóa dữ liệu đầu vào nếu sai
Uư điểm
+Tự build lại những hàm c/c++ (không dùng hàm có sẵn hoặc thư viên)
+Kết hợp nhiều nguồn tài liệu để cho các bạn sinh viên nhập môn kham khảo
Nhược điểm
+Viết code quá dài dẫn đến debug rất khó
+Chưa kết nối hệ cơ sở dữ liệu database,mới lưu trữ tạm thời
